### O que eu tô usando?

- Typescript
- React/ React Hooks
- Redux
- Styled Components

A escolha das stack foi pela familiaridade e tbm pq baseado no desafio eles trabalham melhor juntos, diminuindo o esforço e as linhas de código.

### Para rodar o projeto:

#### `npm install`

#### `npm run start`

ou se preferir:

#### `yarn install`

#### `yarn start`

### Para gerar o build :

#### `npm run buil`

ou

#### `yarn build`
